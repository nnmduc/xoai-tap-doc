# Phase 1: Project Scaffold

## Overview
Bootstrap the React + Vite + TypeScript + Tailwind CSS project at `web-app/`. Configure Vite to serve `assets/` as publicDir. Install all dependencies.

## Requirements
- Functional: `npm run dev` serves app at localhost with hot reload; `npm run build` emits `dist/`
- Non-functional: Vite 5, React 18, TypeScript strict, Tailwind v3, Framer Motion 11

## Architecture
```
web-app/
├── index.html              ← Vite entry, lang="vi", Baloo 2 + Comic Neue preconnect
├── vite.config.ts          ← publicDir: "../assets", base: "./"
├── tailwind.config.ts      ← content: ["./src/**/*.{ts,tsx}", "./index.html"]
├── postcss.config.js
├── tsconfig.json           ← strict: true, paths alias @/*→src/*
├── package.json
├── public/
│   └── stories-manifest.json   ← generated by Phase 2 script
└── src/
    ├── main.tsx
    └── App.tsx             ← placeholder until Phase 6
```

## Implementation Steps

1. **Create `web-app/` directory** and `web-app/public/` subdirectory.

2. **`package.json`** with exact deps:
   ```json
   {
     "name": "xoai-tap-doc-web",
     "private": true,
     "scripts": {
       "dev": "vite",
       "build": "tsc -b && vite build",
       "preview": "vite preview"
     },
     "dependencies": {
       "framer-motion": "^11.0.0",
       "react": "^18.3.0",
       "react-dom": "^18.3.0"
     },
     "devDependencies": {
       "@types/node": "^20.0.0",
       "@types/react": "^18.3.0",
       "@types/react-dom": "^18.3.0",
       "@vitejs/plugin-react": "^4.3.0",
       "autoprefixer": "^10.4.0",
       "postcss": "^8.4.0",
       "tailwindcss": "^3.4.0",
       "typescript": "^5.5.0",
       "vite": "^5.4.0"
     }
   }
   ```

3. **`vite.config.ts`**:
   ```ts
   import { defineConfig } from 'vite'
   import react from '@vitejs/plugin-react'
   import path from 'path'

   export default defineConfig({
     plugins: [react()],
     publicDir: path.resolve(__dirname, '../assets'),
     base: '/',
     resolve: {
       alias: { '@': path.resolve(__dirname, 'src') }
     }
   })
   ```

4. **`tailwind.config.ts`**:
   ```ts
   import type { Config } from 'tailwindcss'
   export default {
     content: ['./index.html', './src/**/*.{ts,tsx}'],
     theme: { extend: {} },
     plugins: [],
   } satisfies Config
   ```

5. **`postcss.config.js`**: `{ plugins: { tailwindcss: {}, autoprefixer: {} } }`

6. **`tsconfig.json`**: strict mode, `jsx: "react-jsx"`, path alias `@/*`.

7. **`index.html`**: `lang="vi"`, Google Fonts preconnect for Baloo 2 + Comic Neue, `<div id="root">`.

8. **`src/main.tsx`**: standard `ReactDOM.createRoot` mount.

9. **`src/App.tsx`**: minimal placeholder `<h1>Xoài Tập Đọc</h1>`.

10. **Install deps**: `cd web-app && npm install`

11. **Smoke test**: `npm run dev` → app loads; `npm run build` → no TypeScript errors.
